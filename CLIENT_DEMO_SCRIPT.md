## Demo Tips

**Preparation:**
1. Have test accounts ready (client account with some data)
2. Pre-populate some bookings, saved caterers, and messages
3. Use realistic test data (names, dates, etc.)
4. Test the demo flow beforehand
5. Have backup screenshots in case of technical issues

**During Demo:**
1. Speak clearly and at a moderate pace
2. Pause briefly after each major action
3. Highlight key features with mouse movements
4. Show both success and error states if possible
5. Keep the demo under 20 minutes total

**Screen Recording Settings:**
1. Use 1920x1080 resolution
2. Record at 30fps minimum
3. Enable cursor highlighting
4. Use a clean browser profile (no extensions visible)
5. Close unnecessary tabs and applications
